Hướng dẫn chạy:
Bước 1: Tải và cài đặt XAMPP
Truy cập trang chủ XAMPP: https://www.apachefriends.org/index.html. 
Chọn phiên bản phù hợp với hệ điều hành (Windows, Linux hoặc macOS).
Chạy tệp cài đặt và làm theo hướng dẫn để hoàn tất quá trình cài đặt.

Bước 2: Tải mã nguồn dự án
Truy cập Mã nguồn dự án "STAGEX" tại: https://github.com/HoaiThu066/FINAL-STAGEX
Tải mã nguồn về máy tính bằng cách nhấn nút "Code" và chọn
"Download ZIP".
Giải nén tệp ZIP vào thư mục htdocs trong thư mục cài đặt XAMPP

Bước 3: Cấu hình cơ sở dữ liệu
Mở XAMPP Control Panel và khởi động Apache và MySQL.
Truy cập phpMyAdmin bằng cách mở trình duyệt và nhập địa chỉ: http://localhost/phpmyadmin. 
Tạo một cơ sở dữ liệu mới với tên: stagex_db
Nhập tệp cấu hình cơ sở dữ liệu từ dự án vào cơ sở dữ liệu mới tạo.

Bước 4: Truy cập ứng dụng. Mở trình duyệt và nhập địa chỉ:
http://localhost/FINAL-STAGEX-main
